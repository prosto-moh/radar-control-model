# Radar Control Model

Математическая модель сопровождения цели угломером РЛС.

Задачи проекта:

1)Составить математическую модель движения цели и РЛС

2)Задать алгоритм управления

3)Научить алгоритм самостоятельно подбирать q-параметры

4)Построить графики ошибок сопровождения по **углу** и по **угловой скорости**
